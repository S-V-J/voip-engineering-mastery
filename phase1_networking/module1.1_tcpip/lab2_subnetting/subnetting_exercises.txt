EXERCISE 1: Basic Voice/Data Separation
═══════════════════════════════════════════════════════════════
Given: 192.168.10.0/24
Requirements:
- 60 IP phones (Voice VLAN 10)
- 100 data devices (Data VLAN 20)
- 20 servers (Server VLAN 30)

SOLUTION:
---------

   - Broadcast: 192.168.10.63
   - Usable: 192.168.10.1 - 192.168.10.62 (62 hosts)
   - VLAN 10 Gateway: 192.168.10.1

2. Data Devices (need 126 usable): 192.168.10.64/25
   - Network: 192.168.10.64
   - Broadcast: 192.168.10.127
   - Usable: 192.168.10.65 - 192.168.10.126 (62 hosts)

   - Phones: 192.168.10.0/26 (62 usable) ✓

     - Network: 192.168.10.64

     - Usable: 192.168.10.65 - 192.168.10.190

EXERCISE 4: Calculate Network Details
═══════════════════════════════════════════════════════════════
Given: 192.168.50.48/28

Calculate:
----------
Subnet Mask: 255.255.255.240
Network Address: 192.168.50.48
Broadcast Address: 192.168.50.63
First Usable: 192.168.50.49
Last Usable: 192.168.50.62
Total Usable Hosts: 14

Binary Calculation:
192.168.50.48 = 11000000.10101000.00110010.00110000
/28 mask      = 11111111.11111111.11111111.11110000
Network bits  = 28, Host bits = 4
Hosts = 2^4 - 2 = 14
